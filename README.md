## Run Locally

Clone the project

```bash
  git clone https://github.com/anhducokok/to-do-react
```

Go to the project directory

```bash
  cd to-do-reactreact
```

Install dependencies

```bash
  npm install
```

Start 

```bash
  npm run run dev
```
